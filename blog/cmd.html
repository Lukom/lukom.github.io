Командний рядок - хитрості і трюки

Розробка CMD, Windows, Автоматика, Лайфхаки, Програмування 2009.05.24 Відредаговано: 2016.05.07

Деякі хитрощі по роботі з командним рядком (КР).



1. Щоб запустити КР натискаємо Win + R (Або просто Win у Windows 7) вводимо cmd і натискаємо Enter

2. Для зручної роботи з КР заходимо у властивості (натискаємо правою кнопкою на заголовку вікна, вибираємо "Властивості") та ставимо галочки для двох опцій:

  • Виділення мишею (Quick edit mode) - дозволяє користуватись мишкою в КР
  • Швидке вставлення (Insert mode) - дозволяє вставляти текст в КР з буферу обміну

Включення цих двох надзвичайно корисних опцій дозволяє:

  • Виділити мишкою довільну область екрану і натиснути Enter, щоб скопіювати виділене у буфер
  • Натиснути правою кнопкою мишки, щоб вставити текст із буферу

3. Кнопки які часто використовуються при роботі з КР:

  • Tab - виводить перший файл у поточній папці, якщо ще раз натиснути - другий, і т.д. Можна набрати першу частину імені файлу, а тоді натиснути Tab, тоді виведеться файл, який починається на запропоновані символи.
  • Up / Down - попередня/наступна команда з історії
  • Right - вивести наступний символ з попередньої команди з історії

4. Спеціальні кнопки:

  • F1 Повторює символи із попередньої команди один за одним (аналог Right)
  • F2 Показує діалог "Копіювати до символу"
  • F3 Показує останню команду
  • F4 Показує діалог "Видалити до символу"
  • F5 Показує попередню команду (аналог Up)
  • F6 Вставляє ^z
  • F7 Показує меню з історією команд
  • F8 Перебирає по черзі команди з історії (Починаючи з останньої)
  • F9 Показує діалог "Введіть номер команди"

5. Щоб виключити КР в англійському віндовсі можна натиснути Alt + Space + C, аналогів для своєї української вісти я не знайшов..
